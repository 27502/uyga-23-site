JavaScript Map va Set

Map va Set haqida tushunchalar, ularning xususiyatlari va kod misollari.

Map nima?

Map — bu JavaScriptda kalit-qiymat juftliklarini saqlash uchun ishlatiladigan ma'lumot tuzilmasi. Unga har qanday turdagi kalitlarni (string, object, number) qo'shish mumkin.

Asosiy xususiyatlari:

  • Kalit-qiymat juftliklarini saqlaydi.
  • Kalit sifatida har qanday turdagi ma'lumotdan foydalanish mumkin.
  • O‘lchamni size xususiyati orqali olish mumkin.
  • Tartibni saqlaydi (qo'shilgan tartibda).

Map misoli:

const map = new Map();
map.set('name', 'Habibullo');
map.set('age', 25);
console.log(map.get('name')); // 'Habibullo'
console.log(map.has('age'));  // true
console.log(map.size);        // 2

Set nima?

Set — bu JavaScriptda takrorlanmas qiymatlar to'plamini saqlash uchun ishlatiladigan ma'lumot tuzilmasi. Unda faqat noyob qiymatlar saqlanadi.

Asosiy xususiyatlari:

  • Takrorlanmas qiymatlarni saqlaydi.
  • Tartibni saqlaydi (qo'shilgan tartibda).
  • O‘lchamni size xususiyati orqali olish mumkin.
  • Faqat qiymatlarni saqlaydi, kalit yo'q.

Set misoli:

const set = new Set();
set.add(1);
set.add(2);
set.add(1); // Takroriy qiymat qo'shilmaydi
console.log(set.size); // 2
console.log(set.has(2)); // true
